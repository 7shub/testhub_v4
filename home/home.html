About Us

Introduction
TestHub is a reliable and secure web-based exam platform designed to streamline the online examination process. It ensures a smooth and efficient experience for teachers, administrators and students.
Key Features
The platform includes automated result generation, which reduces manual effort and increases accuracy. It also supports random question selection and time-bound tests for fair assessments.
User Roles
TestHub provides separate panels for admins, teachers and students. Admins can handle the user joined and quizzes created. Where as teachers can create, manage, and schedule exams. While students can attempt quizzes and view their results.
Security and Authentication
To maintain integrity, TestHub uses face recognition for user authentication. This advanced feature prevents impersonation and ensures that only authorized users can access the tests.
